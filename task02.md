# 두 번째 수시과제

1. 지금까지 학습한 내용을 클래스화 해서 잘 구성(해시 계산을 위한 클래스 포함해서)

2. 아래 세 가지 기능을 구현

    2-1. ".text" 섹션의 md5 해시를 추출해서 "DetectMe.exe" 파일 탐지하는 기능
        --> "DetectMe.exe" 의 md5 해시를 읽는 함수 작성
        --> 스캔 대상 파일의 md5 해시와 대조하는 함수 작성
        --> 검사 수행 완료

    2-2. EntryPointSection 정보를 이용해서 처음 실행되는 코드가 포함된 섹션을 찾고 해당 섹션의 md5 해시를 추출 해서  "DetectMe.exe" 파일 탐지하는 기능
        --> "DetectMe.exe" 의 EntryPointSection 정보를 읽는 함수 작성
        --> EntryPointSection 정보를 이용해서 처음 실행되는 코드가 포함된 섹션을 찾는 함수 작성
        --> 찾은 EntryPointSection 의 md5 해시를 추출하는 함수 작성
        --> 스캔 대상 파일의 md5 해시와 대조하는 함수 작성
        --> 검사 수행 완료

    2-3. pdb 파일 경로에서 md5 해시를 추출해서 "DetectMe.exe" 파일 탐지하는 기능
        
        (강의자료 p.11 참고하면 됨)
        --> "DetectMe.exe" 의 pdb 파일 경로를 읽는 함수 작성
        --> pdb 파일 경로에서 md5 해시를 추출하는 함수 작성
        --> 스캔 대상 파일의 md5 해시와 대조하는 함수 작성
        --> 검사 수행 완료
        https://www.sysnet.pe.kr/2/0/11237
